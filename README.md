📊 Nike Retail Sales Performance Dashboard

📌 Project Overview

This project presents an end-to-end **data analytics and visualization solution** using Power BI to analyze Nike’s retail sales performance.

The dashboard provides a comprehensive view of sales across **regions, product categories, retailers, and time**, enabling stakeholders to quickly identify trends, performance drivers, and growth opportunities.

The objective of this project is to demonstrate how raw retail data can be transformed into **actionable business insights** through effective data modeling and visualization.



 🖼️ Dashboard Preview

![Dashboard](nike-sales-dashboard.png)



🧩 Business Context

Retail organizations like Nike operate across multiple regions and product lines, making it difficult to manually track performance and identify key trends.

Without a centralized dashboard:

* Decision-making becomes slower and less accurate
* High-performing regions or products may go unnoticed
* Seasonal trends are harder to identify
* Strategic planning lacks data support

This dashboard solves these challenges by providing a **single, interactive platform** for performance monitoring.



🎯 Project Objectives

The primary goals of this project were to:

* Analyze overall sales performance
* Identify top-performing regions and retailers
* Evaluate product category performance
* Track sales trends over time
* Build an interactive dashboard for business users



📂 Dataset Description

The dataset used for this project contains retail sales data with the following key fields:

* **Region** – Geographic sales distribution (West, Northeast, Southeast, etc.)
* **Product Category** – Types of Nike products (e.g., Men’s Apparel, Footwear)
* **Retailer** – Sales channels (e.g., One Sports, West Gear)
* **Sales Revenue** – Total revenue generated
* **Units Sold** – Quantity of products sold
* **Month** – Time-based sales tracking

📁 Dataset

🛠️ Data Preparation & Transformation

Before visualization, the dataset underwent several preprocessing steps:

* Data cleaning (handling missing or inconsistent values)
* Standardizing column formats
* Creating calculated measures using **DAX**
* Structuring data for efficient filtering and aggregation

Key measures created:

* Total Sales
* Total Units Sold
* Average Sales per transaction



 📊 Dashboard Features

🔹 Sales by Retailer

These allow users to dynamically explore the data and customize insights.



📈 Key Insights

* 🥇 The **West region** generated the highest revenue (~₦0.28bn), making it the most profitable market
* 👟 **Men’s athletic footwear** is the top-performing product category, indicating strong demand
* 📅 Sales peaked in **December (~₦105M)**, highlighting a clear seasonal trend
* 🏬 **One Sports** is the leading retailer, contributing the highest share of sales



💡 Business Recommendations

Based on the analysis, the following actions are recommended:

* 📍 **Invest more in high-performing regions** (e.g., West) to maximize revenue
* 👟 **Expand inventory and marketing for top products** like athletic footwear
* 🎄 **Leverage seasonal trends** by increasing promotions during peak months
* 🏬 **Strengthen partnerships with top retailers** such as One Sports
